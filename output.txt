## üîÅ 1. **Separate Game Logic from Pygame Rendering**
You must **decouple** the game logic from Pygame-specific rendering.

### ‚úÖ What to Do:
- Create a **pure Python game engine** (no `pygame`, no `pygame.display`, etc.)
- Represent the game state as **JSON-serializable data**:
  - Snake positions
  - Fruit positions
  - Score
  - Game over flag

---

## üß† 2. **Create a Flask Backend (Game API)**

### ‚úÖ Endpoints You Need:
| Endpoint | Method | Purpose |
|---------|--------|---------|
| `/start` | POST | Start a new game (return game ID) |
| `/move` | POST | Send a move (UP/DOWN/LEFT/RIGHT) |
| `/state` | GET | Get current game state as JSON |

### ‚úÖ Example:
```python
# Flask app (simplified)
from flask import Flask, request, jsonify
from game_engine import GameEngine  # your pure Python game logic

app = Flask(__name__)
games = {}

@app.route('/start', methods=['POST'])
def start_game():
    game_id = str(len(games) + 1)
    games[game_id] = GameEngine()
    return jsonify({"game_id": game_id, "state": games[game_id].get_state()})

@app.route('/move', methods=['POST'])
def move():
    game_id = request.json['game_id']
    direction = request.json['direction']
    game = games[game_id]
    game.move(direction)
    return jsonify(game.get_state())

@app.route('/state/<game_id>', methods=['GET'])
def get_state(game_id):
    return jsonify(games[game_id].get_state())
```

## ‚öõÔ∏è 3. **Create a React Frontend**

### ‚úÖ What It Does:
- Sends POST requests to Flask
- Renders the game state as a **grid** (canvas or divs)
- Handles keyboard input (WASD or arrow keys)

### ‚úÖ Example React Component (simplified):
```jsx
import React, { useEffect, useState } from 'react';
import axios from 'axios';

function Game() {
  const [gameId, setGameId] = useState(null);
  const [state, setState] = useState(null);

  useEffect(() => {
    startGame();
  }, []);

  const startGame = async () => {
    const res = await axios.post('/start');
    setGameId(res.data.game_id);
    setState(res.data.state);
  };

  const move = async (dir) => {
    const res = await axios.post('/move', { game_id: gameId, direction: dir });
    setState(res.data);
  };

  useEffect(() => {
    const handleKey = (e) => {
      if (e.key === 'ArrowUp') move('UP');
      if (e.key === 'ArrowDown') move('DOWN');
      if (e.key === 'ArrowLeft') move('LEFT');
      if (e.key === 'ArrowRight') move('RIGHT');
    };
    window.addEventListener('keydown', handleKey);
    return () => window.removeEventListener('keydown', handleKey);
  }, [gameId]);

  if (!state) return <div>Loading...</div>;

  return (
    <div>
      <h2>Score: {state.score}</h2>
      <div style={{ display: 'grid', gridTemplateColumns: `repeat(${state.grid_size}, 30px)` }}>
        {state.grid.map((row, y) =>
          row.map((cell, x) => (
            <div
              key={`${x}-${y}`}
              style={{
                width: 30,
                height: 30,
                backgroundColor:
                  cell === 'snake' ? 'green' :
                  cell === 'fruit' ? 'red' :
                  cell === 'poison' ? 'purple' : '#eee',
                border: '1px solid #ccc'
              }}
            />
          ))
        )}
      </div>
    </div>
  );
}

export default Game;
```
